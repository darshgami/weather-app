# 🌦️ Weather App

A simple weather web application built using **Django**, **HTML**, **CSS**, and **JavaScript**.
---
## 🚀 Features

* Get real-time weather by city name
* Shows temperature and weather condition
* Simple and clean user interface
---
## 🛠️ Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, JavaScript
* API: OpenWeather
---
## ⚙️ How to Run
```bash
git clone https://github.com/your-username/django-weather-app.git
cd django-weather-app
pip install django requests
python manage.py runserver
```
Open in browser:
```
http://127.0.0.1:8000/
```
---
## 🔑 API Setup
Add your API key in `views.py`:

```python
API_KEY = "your_api_key"
```
